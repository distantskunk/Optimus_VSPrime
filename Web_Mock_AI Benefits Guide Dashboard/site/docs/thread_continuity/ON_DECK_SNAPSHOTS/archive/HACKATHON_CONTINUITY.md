1. Project Identity
AI Benefits Explainer
Build a guided AI-assisted member experience that:

Interprets member questions
Returns structured answers
Provides a clear next action
Personalizes responses based on member context
Escalates appropriately when confidence is low or issues are out of scope

The project objective is a guided experience rather than a generic chatbot experience. The expected interaction pattern is Interpret → Answer → Guide → Escalate.+2
